The Three Layers

Infrastructure (Shared)
Agent definitions, MCP server code, hooks, CLAUDE.md templates, webhook server. Shared via this repo.
Configuration (Per Person)
API keys, OAuth tokens, environment variables, personal settings. Set up during onboarding.
State (Local/Ephemeral)
Conversation history, auto-memory, scratchpad files. Generated as you work. Not shared.
