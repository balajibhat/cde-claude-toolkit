Quick Start

Zero to operational in under 30 minutes. Follow these steps in order.

Prerequisites: You need Claude Code installed (see Installing Claude Code) and a Claude account with an active subscription.

The Steps

  1. Install Claude Code
    Follow the installation guide. Make sure both the CLI and VS Code extension are working. Verify by running claude --version in your terminal.
  2. Clone this repo
    git clone https://github.com/balajibhat/cde-claude-toolkit.git
  3. Copy the starter CLAUDE.md
    Copy starter/CLAUDE.md from this repo to your home directory's .claude/CLAUDE.md:
    # Windows copy cde-claude-toolkit\starter\CLAUDE.md %USERPROFILE%\.claude\CLAUDE.md # Mac cp cde-claude-toolkit/starter/CLAUDE.md ~/.claude/CLAUDE.md
  4. Install safety hooks
    These hooks protect your machine from dangerous commands. From the cde-claude-toolkit directory:
    # Windows starter\hooks\install-hooks.bat # Mac / Linux / Git Bash bash starter/hooks/install-hooks.sh
    See Hooks & Safety for details and manual setup.
  5. Set up MCP servers
    Run the MCP setup script to add the core servers automatically:
    # Windows starter\setup-mcp.bat # Mac / Linux / Git Bash bash starter/setup-mcp.sh
    Or follow the MCP Servers guide to add them manually.
  6. Set up authentication
    Follow the Authentication guide for any MCP servers that need API keys or OAuth tokens.
  7. Verify your setup
    Run the verification script to confirm everything is configured:
    # Windows starter\verify-setup.bat # Mac / Linux / Git Bash bash starter/verify-setup.sh
    You should see all PASS results. Fix any FAILs and re-run.
