MCP Servers

MCP (Model Context Protocol) servers are plugins that give Claude new abilities. Here's what CDE uses and how to set each one up.

You don't need all of these. Start with Playwright (zero config), then add others as you need them.

How to Add an MCP Server

MCP servers are configured in your Claude Code settings. You can add them via the CLI or by editing the settings file directly.

Via CLI (easiest)

# Add Playwright (zero config) claude mcp add playwright -- npx @playwright/mcp@latest # Add Supadata (zero config) claude mcp add supadata -- npx @supadata/mcp # Add Stitch (zero config) claude mcp add stitch -- npx @_davideast/stitch-mcp

Via Settings File

Edit ~/.claude/settings.json and add to the mcpServers object:

{ "mcpServers": { "playwright": { "command": "npx", "args": ["@playwright/mcp@latest"] }, "supadata": { "command": "npx", "args": ["@supadata/mcp"] } } }

Server Details

Playwright (Browser Automation)

Navigate to URLs, take screenshots, click elements, fill forms, read page content. Essential for QA, research, and web scraping.

# Example: take a screenshot of a page "Navigate to balajibhat.github.io/cde-services and take a screenshot" # Example: QA a deployed page "Check if the sidebar links on our proposal are all working"

Supadata (Transcripts + Scraping)

Pull YouTube video transcripts and scrape web pages. Great for research and content creation.

# Example "Get the transcript from this YouTube video: [URL]"

Google Drive (Docs/Sheets Access)

Read Google Docs and Sheets directly. Used by agents to access the Master Brain Doc, content pipeline, and client data. Requires OAuth setup — see Authentication.

Nano Banana (Image Gen/Edit)

Generate and edit images using Gemini. Requires a Gemini API key — configure it using the configure_gemini_token tool after adding the server.

claude mcp add nano-banana -- npx nano-banana-mcp
