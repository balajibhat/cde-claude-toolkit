Troubleshooting

Common issues and how to fix them.

Claude Code Won't Start

SymptomLikely CauseFix
claude: command not foundNot installed or not in PATHRun npm install -g @anthropic-ai/claude-code
Authentication errorNot logged in or subscription expiredRun claude login or check your subscription at claude.ai
VS Code extension not showingExtension not installed or disabledCheck Extensions panel, search "Claude Code"

MCP Server Issues

SymptomLikely CauseFix
"Server not connected"Server crashed or wasn't startedRestart Claude Code. Check claude mcp list
Google Drive auth expiredOAuth tokens expiredDelete ~/.claude/.gdrive-server-credentials.json and re-auth
Playwright can't open browserNo browser installed or permissions issueRun npx playwright install chromium
Image gen failsMissing or invalid Gemini API keyCheck GEMINI_API_KEY env var or reconfigure

Hook False Positives

Sometimes hooks block safe operations because they match a dangerous pattern.

SymptomLikely CauseFix
Blocked on word "format" in Python codeHook matches system command keywords in inline codeWrite the script to a file first, then run the file
Blocked on a safe git commandCommand matches destructive git patternRun the command manually in your terminal
If you keep hitting false positives on a specific pattern, tell Balaji so the hook can be tuned. Don't try to modify hooks yourself.

General Tips

Restart fixes most things
Close and reopen Claude Code. MCP servers reconnect on startup.
Check your Node version
Claude Code needs Node.js 18+. Run node --version to check.
Clear npm cache
If MCP servers fail to start: npm cache clean --force
Ask Claude
Seriously — describe your problem to Claude. It's good at debugging itself.
