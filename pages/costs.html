Costs & Billing

Exactly what costs money, what doesn't, and what's optional. No surprises.

The golden rule: Your keys = your cost. Nothing in this toolkit charges anyone else's account. You control what you pay for by choosing which services to activate. You can use Claude Code with zero optional add-ons and still be highly effective.

The Only Required Cost

To use Claude Code, you need exactly one thing: a Claude subscription. That's it. Everything else on this page is optional.

PlanMonthly CostWhat You Get
Claude Max $100/mo Daily use — research, content writing, client work, light coding. Enough for most team members.
Claude Max (Heavy) $200/mo Unlimited-feel usage — heavy coding, running multiple agents, complex multi-step tasks. You'll know if you need this because you'll hit rate limits on the $100 plan.
Start with $100/mo. That covers a full workday of active Claude usage. Upgrade to $200 only if you're consistently hitting rate limits. You can change plans anytime.

Optional Upgrades (The Super Max Deluxe Abilities)

These are power-user tools that unlock extra capabilities. Each one requires its own API key — your key, on your account, billed to you. None of these are required. Think of them as unlockable abilities you can opt into when you need them.

None of these charge CDE or anyone else. When you create an API key, it's tied to YOUR account. You set your own spending limits. You can revoke it anytime.

What it costsRequires a Gemini API key from Google AI Studio. Free tier gives you ~50 images/day. Paid tier is ~$0.01–0.05 per image.

API-Powered Claude (Pay-Per-Use)

Alternative to Claude Max subscription
What it doesInstead of a flat monthly subscription, you pay per conversation. Some people prefer this for lighter or unpredictable usage.

Spending Controls

If you activate any paid API keys, here's how to stay in control:

Gemini API
Google AI Studio lets you set daily quotas and spending caps. The free tier has built-in rate limits that prevent accidental overuse.
Anthropic API
Anthropic's console lets you set monthly spending limits. Set a hard cap before you start using it. It will stop when you hit it.
Claude Max
Flat rate. No usage surprises. If you hit the rate limit, Claude just slows down — it doesn't charge you more.
Bottom line: If you just get a $100/mo Claude Max subscription and use the free MCP servers (Playwright, Supadata, Google Drive), your total cost is $100/mo. Everything else is opt-in.
