Installing Claude Code

Get Claude Code running on your machine. Three options depending on how you work.

Option 1: VS Code Extension (Recommended)

Best for most people. Claude lives right inside your editor.

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
  3. Search for "Claude Code" by Anthropic
  4. Click Install
  5. Open Claude with Ctrl+L / Cmd+L
  6. Sign in with your Anthropic account when prompted

Option 2: CLI (Terminal)

For power users who prefer the command line.

# Install via npm (requires Node.js 18+) npm install -g @anthropic-ai/claude-code # Verify installation claude --version # Start a session claude
If you don't have Node.js, download it from nodejs.org. Get the LTS version.

Option 3: Desktop App

Standalone app, no VS Code required.

  1. Go to claude.ai/download
  2. Download for your OS (Windows or Mac)
  3. Install and sign in
