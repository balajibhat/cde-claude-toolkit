Agents

CDE's AI agents — specialized Claude instances that handle specific jobs automatically or on demand.

Agents are Claude Code sessions with tailored instructions (via CLAUDE.md) and specific tools enabled. Some run automatically via webhooks; others you trigger manually.

Using Agents Day-to-Day

Most agents run automatically. For manual agents, you typically just open Claude Code and tell it what you need. The CLAUDE.md file loaded in your session handles the rest.

You don't need to memorize agent names or commands. Just describe what you want to do and Claude will use the right tools and follow the right processes based on your CLAUDE.md configuration.
